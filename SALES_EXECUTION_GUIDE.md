## 📋 完整销售流程

### 阶段1：需求发现（每日执行）

**目标：** 每日发现≥10个潜在需求

**执行步骤：**

1. **知乎扫描**
   - 搜索关键词：数据分析、PPT制作、Excel、Python、求助
   - 筛选标准：
     - 发布时间≤7天
     - 问题清晰，有具体需求
     - 评论数少（竞争小）
   - 记录到需求池

2. **V2EX扫描**
   - 搜索节点：freelancer、python、web
   - 筛选标准：
     - 有明确预算
     - 需求匹配我的能力
     - 发帖时间≤3天
   - 记录到需求池

3. **CSDN扫描**
   - 搜索：外包、求助、开发
   - 筛选标准：
     - 技术栈匹配
     - 预算合理
     - 发帖时间≤5天
   - 记录到需求池

4. **Boss直聘扫描**
   - 搜索：远程、兼职、Python、数据分析
   - 筛选标准：
     - 远程/兼职工作
     - 技能匹配
     - 薪资合理
   - 记录到需求池

**输出：** `demands.json` - 每日需求列表

---

### 阶段2：需求匹配与评估（发现后30分钟内）

**目标：** 每个需求在30分钟内完成评估

**评估清单：**

```
□ 需求类型（数据分析/PPT/Excel/网站/脚本）
□ 需求复杂度（简单/中等/困难）
□ 预估工时（小时）
□ 基础报价（使用定价算法）
□ 初步解决方案
□ 相似案例（如有）
```

**定价算法：**

```
基础价格 = 根据服务类型查表

复杂度系数：
  简单: 0.7
  中等: 1.0
  困难: 1.3

紧急度系数：
  不急: 0.9
  一般: 1.0
  紧急: 1.5

最终报价 = 基础价格 × 复杂度系数 × 紧急度系数
```

**示例：**
- 需求：数据分析报告
- 基础价格：¥2000
- 复杂度：中等（×1.0）
- 紧急度：一般（×1.0）
- 最终报价：¥2000

**输出：** 需求评估记录

### 阶段3：主动联系（评估后立即）

**目标：** 每日联系≥5个潜在客户

**联系优先级：**

1. **高优先级**
   - 有明确预算
   - 需求紧急
   - 竞争小（回复少）

2. **中优先级**
   - 需求清晰
   - 预算合理
   - 技能完全匹配

3. **低优先级**
   - 需求模糊
   - 预算偏低
   - 竞争激烈

**联系方式选择：**

| 平台 | 首选方式 | 备选方式 |
|------|---------|---------|
| 知乎 | 私信 | 评论 |
| V2EX | 私信 | 回复 |
| CSDN | 私信 | 评论 |
| GitHub | Issue评论 | - |
| Boss直聘 | 立即沟通 | - |

**联系消息模板：**

**通用模板：**
```
你好！看到你的【需求类型】需求，我可以提供专业帮助。

我提供的服务：
- 数据分析：¥500-5000/份
- PPT制作：¥200-3000/份
- Excel开发：¥300-2000/个
- 网站开发：¥1000-20000/站

针对你的需求，初步分析：
- 解决方案：简要描述
- 预估工时：X天
- 参考报价：¥XXX

我可以先免费评估需求，提供详细方案。如果有兴趣请回复。

我的案例：https://ourlyg.com
联系方式：contact@ourlyg.com
```

**关键原则：**
- ✅ 第一句话就表明能力
- ✅ 提供具体报价范围
- ✅ 免费评估，降低门槛
- ✅ 案例链接，增强信任
- ❌ 不要长篇大论
- ❌ 不要过度营销

**输出：** 联系记录（CRM）

---

### 阶段4：深度沟通与报价（收到回复后24小时内）

**目标：** 快速成交，报价转化率≥50%

**沟通流程：**

**步骤1：确认详细需求**
```
询问内容：
1. 具体的需求描述（越详细越好）
2. 数据来源和格式
3. 截止时间
4. 预算范围
5. 特殊要求
```

**步骤2：提供具体报价**
```
报价单结构：
- 项目明细（服务项、数量、单价）
- 总价
- 交付时间
- 工作流程
- 付款方式
- 质保（免费修改）
```

**步骤3：展示案例**
```
提供：
- 类似项目的案例
- 客户评价（如有）
- 工作流程说明
```

**报价原则：**
- ✅ 价格透明，明细清晰
- ✅ 有竞争力，不低于市场价
- ✅ 新客户优惠（首单立减15%）
- ✅ 不满意免费修改
- ❌ 不要随意降价
- ❌ 不要夸大承诺

**成交信号：**
- ✅ 客户询问工作流程
- ✅ 客户要求提供方案
- ✅ 客户确认预算
- ✅ 客户询问付款方式

**流失信号：**
- ❌ 客户一直不回复
- ❌ 客户说"再考虑"
- ❌ 客户要求大幅降价
- ❌ 客户已有其他供应商

**输出：** 沟通记录 + 报价单

---

### 阶段5：执行与交付（按里程碑）

**目标：** 高质量交付，客户满意度≥95%

**项目执行流程：**

**里程碑1：30% - 需求确认与大纲（第1天）**
```
□ 确认详细需求
□ 提供项目大纲
□ 确认设计方案
□ 收集素材和数据
```

**里程碑2：60% - 核心功能完成（第N天）**
```
□ 完成核心功能
□ 内部测试
□ 提供预览版本
□ 收集反馈
```

**里程碑3：90% - 细节优化（第N+1天）**
```
□ 细节调整
□ 优化用户体验
□ 完成所有功能
□ 最终测试
```

**里程碑4：100% - 最终交付（第N+2天）**
```
□ 发送最终交付物
□ 提供使用说明
□ 确认客户满意
□ 收集反馈
□ 收款
```

**交付物清单：**
- 最终成果文件
- 源文件（如需）
- 使用说明/文档
- 质保承诺（7天内免费修改）

**交付原则：**
- ✅ 按时交付，不拖延
- ✅ 质量第一，零错误
- ✅ 沟通及时，每日汇报
- ✅ 不满意免费修改
- ❌ 不要低于预期
- ❌ 不要遗漏需求

**输出：** 项目文档 + 交付物

---

### 阶段6：跟进与复购（交付后持续）

**目标：** 复购率≥30%

**跟进节奏：**

**交付后第1天：**
- 确认客户收到交付物
- 询问是否满意
- 提供使用指导

**交付后第7天：**
- 跟进使用情况
- 收集反馈
- 主动提出优化建议

**交付后第30天：**
- 询问是否有新需求
- 推荐新服务
- 建立长期合作关系

**复购激励：**
- 长期合作享8折
- 推荐朋友返现¥50
- 季度套餐享7折
- 年度套餐享6折

**输出：** 客户记录 + 复购记录
