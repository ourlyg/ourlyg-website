# 理易港

> 让理财更简单，让投资更智慧

## 网站简介

理易港（Ourlyg.com）是一个专注于个人理财和投资教育的平台，致力于通过工具、测评和资讯，帮助用户做出更明智的财务决策。

---

## 核心功能

### 📊 理财工具

我们提供实用的理财计算工具，帮助用户规划财务：

- **投资风险评估** - 通过10道科学问卷，评估您的风险偏好类型（保守型/稳健型/进取型/激进型），并推荐适合的理财产品
- **房贷计算器** - 支持等额本息和等额本金两种还款方式，计算月供、利息总额和还款明细
- **养老规划计算器** - 帮助用户规划退休储备，计算达到目标养老金额所需的月储蓄
- **收益计算器** - 计算投资复利收益，展示不同时间周期的财富增长

### 📰 产品测评

我们提供独立客观的金融产品测评：

- **券商测评** - 华泰证券等主流券商的产品和服务分析
- **银行理财** - 招商银行朝朝宝等理财产品的收益对比
- **货币基金** - 支付宝余额宝等货币基金的费率和收益分析
- **平台对比** - 不同理财平台的优缺点、费率对比

测评特点：
- 数据驱动，客观中立
- 费率透明，收益清晰
- 适合不同风险偏好人群

### 🏪 营业网点查询

- 查找附近的银行、券商营业网点
- 基于地理位置，提供联系方式和营业时间
- 支持多种金融机构类型
