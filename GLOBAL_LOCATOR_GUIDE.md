# 客户全网定位系统 - 操作手册

## 📖 系统概述

**客户全网定位系统**是Ourlyg AI的核心竞争力，用于绕过平台限制，直接触达客户。

### 核心价值
- ✅ **不受平台限制**：不依赖平台私信功能
- ✅ **长期客户关系**：建立独立客户数据库
- ✅ **跨平台复用**：一个客户，多平台联系
- ✅ **高转化率**：直接沟通，缩短决策链路

### 适用场景
- 在知乎/V2EX/CSDN发现需求
- 在Boss直聘看到招聘需求
- 在GitHub看到Issue求助
- 在小红书看到合作需求

---

## 🚀 快速开始

### 1. 发现需求

从各平台获取需求信息：

```python
from global_locator import CustomerGlobalLocator

locator = CustomerGlobalLocator()

result = locator.locate_and_contact(
    platform="知乎",
    content="需求内容",
    service_type="数据分析",
    price="1500-3000"
)
```

### 2. 查看结果

系统会自动：
- ✅ 搜索引擎聚合搜索
- ✅ 跨平台社交媒体搜索
- ✅ 企业信息查询
- ✅ 建立客户档案
- ✅ 生成联系消息

### 3. 主动联系

根据系统提供的联系方式，选择最佳方式联系：
1. **邮箱**（优先级最高）：发送邮件
2. **微信**：添加好友并发送消息
3. **电话**：打电话（需谨慎，避免打扰）
4. **平台私信**（备选）：如果其他方式不可用

## 🔍 定位方法详解

### 方法1：搜索引擎聚合

**原理：** 用客户昵称在各大搜索引擎搜索

**搜索关键词：**
```
"{昵称}" 邮箱
"{昵称}" 微信
"{昵称}" github
"{昵称}" 博客
"{昵称}" 个人网站
```

**目标平台：**
- Google
- 百度
- Bing

**提取信息：**
- 邮箱地址
- 微信号
- 电话号码
- 个人网站
- GitHub账号
- LinkedIn账号

---

### 方法2：社交媒体跨平台搜索

**原理：** 在其他社交平台搜索同一用户

**目标平台：**
- 微博（site:weibo.com）
- 小红书（site:xiaohongshu.com）
- B站（site:bilibili.com）
- Twitter（site:twitter.com）
- 知乎（site:zhihu.com）
- CSDN（site:csdn.net）

**提取信息：**
- 个人资料描述中的联系方式
- 网站链接
- 邮箱地址
- 微信号

---

### 方法3：企业信息查询

**原理：** 针对企业客户，查询官方联系方式

**查询方法：**
1. 搜索企业官网
2. 查找"关于我们"页面
3. 查找"联系我们"页面
4. 查找招聘信息（HR邮箱）
5. 查找新闻稿（负责人联系方式）

**提取信息：**
- 官网URL
- 企业邮箱
- 联系电话
- 公司地址
- 负责人姓名

---

### 方法4：网站Whois查询

**原理：** 查询客户拥有的域名，提取注册信息

**提取信息：**
- 注册邮箱
- 联系电话
- 注册人姓名
- 注册地址

## 📝 工作流程

### 阶段1：需求提取
```
任务：
- 从平台提取客户基本信息
- 提取昵称/用户名/公司名
- 记录需求描述

输出：客户基本信息
```

### 阶段2：全网搜索
```
任务：
- 用昵称进行搜索引擎聚合搜索
- 在社交媒体平台跨平台搜索
- 查询企业官网信息
- 查询网站Whois信息（如果有）

输出：潜在联系方式列表
```

### 阶段3：信息验证
```
任务：
- 验证邮箱格式
- 验证URL有效性
- 去重无效信息
- 优先级排序

输出：有效联系方式列表
```

### 阶段4：档案建立
```
任务：
- 整合所有信息
- 建立客户档案
- 保存到数据库

输出：完整客户档案
```

### 阶段5：主动联系
```
任务：
- 选择最佳联系方式
- 生成专业联系消息
- 发送并记录
- 24小时跟进

输出：联系记录 + 后续行动
```

## 🔐 隐私和合规

### ✅ 允许的操作
- 仅使用公开渠道信息
- 仅使用合法手段获取信息
- 尊重客户隐私
- 遵守GDPR等法规

### ❌ 禁止的操作
- 使用黑客手段获取信息
- 购买非法数据
- 骚扰客户
- 侵犯隐私
- 未经同意发送垃圾邮件

### 最佳实践
1. **个性化消息**：针对客户需求定制消息，不要群发
2. **尊重时间**：避免在深夜或周末联系
3. **提供价值**：消息中提供有用信息，不要纯推销
4. **允许退订**：如果客户不感兴趣，不再联系
5. **保护数据**：客户信息仅用于业务联系，不泄露给第三方
