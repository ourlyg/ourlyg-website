## 群聊进群策略（狗子建议）

### 第一步：进群前准备
```
1. 准备自我介绍：
   - 我是做数据分析/Python自动化的
   - 能处理Excel清洗、可视化、爬虫采集
   - 有外包经验，交付快、改稿不墨迹
   
2. 准备作品案例（脱敏版）：
   - 销售日报自动化
   - 用户留存分析报告
   - 竞品价格监控
```

### 第二步：进群后第一周
```
Day 1：发自我介绍，不主动推销
Day 2-3：看看群里在聊什么，偶尔插话
Day 4-5：有人问技术问题，免费解答
Day 6-7：混脸熟，让大家知道你能干活
```

### 第三步：接单模式
```
模式1：被动等待
   - 群里有人发"有单子"时，第一时间响应
   - 问清楚需求，给出报价和时间
   
模式2：主动展示
   - 每周发一次"本周案例：XXX项目"
   - 带脱敏后的截图，证明你真的能干
   
模式3：私聊转化
   - 看到有人在群里问技术问题，私聊帮忙
   - 解决后顺便问"还有其他需求吗"
```

---

## 警惕事项

1. **不要进群就发广告** —— 会被踢，先混脸熟
2. **报价不要太低** —— 影响你长期收益，参考猪八戒文案里的定价
3. **先确认需求再接单** —— 避免返工，需求越清楚越少麻烦
4. **留好证据** —— 聊天记录、付款记录、需求文档
