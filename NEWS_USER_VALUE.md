# 📰 财经新闻页面 - 用户价值说明

## 🎯 核心价值

### 为用户带来什么
1. **及时性** - 每天 3 次自动更新（6 点、12 点、18 点）
2. **权威性** - 来自 5 大权威财经网站
3. **智能性** - AI 去重，避免重复内容
4. **便利性** - 一站式汇总，无需多网站切换
5. **个性化** - 支持分类筛选和关键词搜索

### 主要功能

#### 1. 新闻聚合
- 实时聚合新浪财经、网易、腾讯、东方财富、第一财经的头条新闻
- 每天 3 次自动更新
- 最新新闻优先显示

#### 2. 智能分类
- 政策新闻
- 股市行情
- 基金动态
- 新股申购
- 外汇期货
- 理财产品

#### 3. 高级筛选
- **按源筛选** - 选择感兴趣的财经网站
- **按分类筛选** - 关注特定的新闻类别
- **关键词搜索** - 快速找到相关新闻

#### 4. 用户友好
- 响应式设计，支持手机、平板、电脑
- 相对时间显示（"刚刚"、"1小时前"等）
- 一键跳转到原文（点击标题跳转）
- 深色/浅色主题自动适配
